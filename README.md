# Khoj

##Currently under progress.

A search engine built from scratch to learn and implement Information Retrieval concepts, starting with keyword search and gradually progressing to semantic search, hybrid retrieval, and Retrieval-Augmented Generation (RAG).


## Planned Features

* TF-IDF
* Cosine Similarity
* Vector Space Model
* Semantic Search
* Sentence Embeddings
* Vector Database (FAISS)
* Hybrid Search
* Retrieval-Augmented Generation (RAG)
* FastAPI Backend
* React Frontend


## Goal

The objective of this project is to understand how search engines work by implementing each retrieval technique from scratch instead of relying on existing libraries. The project begins with classical Information Retrieval techniques and progressively evolves into a modern semantic search system powered by embeddings and RAG.
