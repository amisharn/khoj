What is a corpus?

- A huge collection of raw unstructured text or speech data.

What is a document?

- Can be a sentence, a paragraph, a file basically distinct amount of text.
- Smallest retrievable unit of text, meanong changes depending on context.

What is tokenization?

- Breaking down a document into a bunch of tokens.

Why do we need an inverted index?

- For quick retrieval. Reading every document is impossible during every search.

Why is it called "inverted"?

- Cause it maps word to documents.
